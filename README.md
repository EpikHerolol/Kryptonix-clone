# React + Vite

A front-end clone of **Kryptonix** — a PC parts marketplace. Built as a personal project to practice frontend development through real-world UI replication. Recently migrated from vanilla HTML/CSS/JS to **React + Vite**.

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React 19 | Component-based UI |
| Vite | Build tool & dev server |
| CSS3 | Styling & layout |
| JavaScript (ES6+) | Interactivity & state management |

---

## ✅ Features

- Component-based architecture with React
- Shopping cart — add, remove, and subtotal calculation
- Slide-in cart sidebar with overlay
- Login dropdown with click-outside close
- Product card hover animations
- Video banner and image banners
- Responsive navbar with search bar
- Custom font and branded styling

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/kryptonix.git

# Navigate into the project folder
cd kryptonix

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open `http://localhost:5173` in your browser.

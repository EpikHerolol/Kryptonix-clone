# Kryptonix 🖥️

A front-end clone of **Kryptonix** — a PC parts marketplace. Built as a personal project to practice HTML, CSS, and JavaScript through real-world UI replication.

> ⚠️ This is a work in progress. JavaScript functionality is not yet implemented.

---

## 🔗 Inspiration / Reference

This project is inspired by the original Kryptonix website:

👉 https://kryptronix.in/

The goal was to replicate the UI and improve understanding of real-world front-end structure and design patterns.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure & markup |
| CSS3 | Styling & layout |
| JavaScript | Interactivity _(coming soon)_ |

---

## ✅ Current Features

- Structured HTML layout matching the Kryptonix site
- CSS styling with responsive design considerations
- Static product/marketplace UI components
- Assets organised into `images/` and `videos/` directories
